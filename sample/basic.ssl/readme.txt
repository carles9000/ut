Welcome to UT SSL
=================
This is the basic structure of a web app project with UT and SSL
Ready for use !

You have to generate your certificates, otherwise the application will not work
Look for more information at https://carles9000.github.io/ and look for SSL

https://carles9000.github.io/index_doc_en.html?search=ssl 

In the \dll folder you have the different dlls that you will need if you don't have 
them installed on your windows\system32. If you install them in the exe directory 
they will also will execute correctly.

NOTE: In order to correctly use UT with Curl and SSL, you must use 
      the latest version of Harbour (2025) which updates to the latest 
	  versions of Curl and OpenSSL
